# MCP Azure Tools

Libreria multi-modulo Spring Boot MCP per Azure Cloud (ARM, Graph, Key Vault, Storage, App Service, AKS). Maven Central: `io.github.massimilianopili:mcp-azure-all`.

## Build

```bash
/opt/maven/bin/mvn clean compile
/opt/maven/bin/mvn clean install -Dgpg.skip=true
/opt/maven/bin/mvn clean deploy
```

Java 17+. Maven: `/opt/maven/bin/mvn`.

Tool, struttura, configurazione e dipendenze: vedi [README.md](README.md).
Ricerca semantica: `embeddings_search_docs("mcp-azure-tools")`
